# Amazon Reviews Sentiment Analysis

## Project Overview
This project performs Sentiment Analysis on Amazon product reviews using Natural Language Processing (NLP) techniques in Python.

The project analyzes customer reviews and classifies them into:
- Positive
- Negative
- Neutral

The results are visualized using graphs and charts for better understanding of customer sentiment.

---

## Technologies Used
- Python
- Pandas
- TextBlob
- Matplotlib

---

## Features
- Review sentiment classification
- Natural Language Processing (NLP)
- Sentiment visualization using bar graph
- Sentiment distribution using pie chart
- Customer review analysis

---

Amazon Reviews Dataset from Kaggle
